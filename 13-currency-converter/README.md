# Currency Converter

Short description

A simple currency conversion UI. May use fixed sample rates or fetch live rates from a public API.

Files

- `index.html` — entry point
- `script.js` — conversion logic
- `style.css` — styles

How to run

Open `index.html` in a browser. If the project fetches rates from an external API, serve it from a local server and check `script.js` for API details.

Notes

- Live conversion requires an API (many free and paid providers exist). Be mindful of rate limits.
