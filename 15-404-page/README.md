# 404 Page

Short description

A standalone 404 error page design. Useful as a template for websites that need a friendly not-found page.

Files

- `index.html` — the 404 page
- `style.css` — styles

How to run

Open `index.html` in a browser or include the page in your site's 404 routing.

Notes

- Enhance with animations or a search/links back to the main site to improve UX.
