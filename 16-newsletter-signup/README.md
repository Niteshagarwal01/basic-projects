# Newsletter Signup

Short description

A styled newsletter signup form UI. By default it's a front-end only demo and does not include a backend to process subscriptions.

Files

- `index.html` — entry point
- `style.css` — styles

How to run

Open `index.html` in a browser.

Notes

- To collect real subscriptions, integrate with an email provider (Mailchimp, ConvertKit) or use serverless functions to store emails securely.
