# Color Palette Generator

Short description

A small tool to generate and copy color palettes. Useful to explore color combinations and demonstrate clipboard/copy interactions and color inputs.

Files

- `index.html` — entry point
- `script.js` — palette generation logic
- `style.css` — styles

How to run

Open `index.html` in a browser or serve the folder with a static server.

Notes

- Uses modern clipboard APIs in supported browsers for copy-to-clipboard functionality.
