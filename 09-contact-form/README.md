# Contact Form (UI)

Short description

A contact form UI example. This project shows form layout and styling. By default it does not include a backend to submit messages.

Files

- `index.html` — entry point
- `style.css` — styles

How to run

Open `index.html` in a browser.

Notes

- To make the form functional, wire it to a backend endpoint or use a service like Formspree, Netlify Forms, or a serverless function.
