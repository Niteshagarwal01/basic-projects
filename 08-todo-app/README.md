# Todo App

Short description

A classic todo list app demonstrating add, edit, complete, and remove operations. Usually demonstrates persistence via `localStorage`.

Files

- `index.html` — entry point
- `script.js` — todo app logic
- `style.css` — styles

How to run

Open `index.html` in a browser or serve the folder with a static server.

Notes

- Try adding a couple of items and refreshing to verify persistence behavior.
