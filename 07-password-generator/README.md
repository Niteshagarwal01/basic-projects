# Password Generator

Short description

A utility to generate random passwords with options such as length and character sets (lowercase, uppercase, numbers, symbols).

Files

- `index.html` — entry point
- `script.js` — password generation logic
- `style.css` — styles

How to run

Open `index.html` in a browser.

Notes

- Check entropy and avoid recommending insecure defaults. Use the generator for demo purposes only.
